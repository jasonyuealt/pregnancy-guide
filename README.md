# 孕期指南

帮助准爸爸了解孕期每周该做什么、买什么、注意什么的任务清单工具。

## 功能特点

### 首页 - 任务清单

- **按周查看**：左右切换查看不同孕周的内容
- **四类内容**：
  - 待办事项：本周需要做的事
  - 购物清单：本周建议购买的物品
  - 注意事项：本周需要关注的事项
  - 为老婆做：贴心小事建议
- **来源标注**：每条内容标注来源（AI / 小红书 / 自定义）
- **可编辑**：所有内容可添加、编辑、删除
- **任务勾选**：完成的事项可勾选

### AI 功能

- 自动生成每周推荐内容
- 分析小红书内容并提取有用信息
- 智能分类导入的内容

### 设置

- 配置末次月经日期或预产期
- 自动计算当前孕周

## 技术栈

- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **状态管理**: Zustand (持久化存储)
- **AI**: Qwen 大模型
- **图标**: Lucide React

## 数据结构

```typescript
// 每条内容
interface WeekItem {
  id: string;
  type: 'todo' | 'shopping' | 'tip' | 'forWife';
  content: string;
  source: 'ai' | 'xiaohongshu' | 'user';
  completed: boolean;
  week: number;
  note?: string;
}
```

## 页面结构

```
/           # 首页 - 任务清单
/settings   # 设置 - 预产期配置
```

## 本地开发

```bash
# 安装依赖
yarn install

# 启动开发服务器
yarn dev
```

## 设计原则

1. **简洁聚焦**：一页展示当前周所有内容
2. **来源透明**：每条内容标注来源，建立信任
3. **可编辑**：用户可完善、修改、删除内容
4. **任务化**：完成后可勾选，有成就感
