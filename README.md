# 孕期指南 (Pregnancy Guide)

帮助准爸爸/准妈妈了解孕期每周该做什么、注意什么、买什么的简洁工具。

## 核心功能

| 页面 | 路由 | 说明 |
|------|------|------|
| 首页 | `/` | 当前孕周进度、本周待办、本周购买、重点提醒 |
| 孕期时间轴 | `/timeline` | 以周为单位查看：胎儿发育、注意事项、产检、建议购买 |
| 购物清单 | `/shopping` | 完整购物清单管理 |
| 导入管理 | `/import` | 管理从小红书导入的内容 |

## 特色功能

- **全局快速导入**：右下角悬浮按钮，随时粘贴小红书链接，AI 解析并建议放置位置
- **AI 内容生成**：根据孕周自动生成专业指导内容
- **简洁清爽**：信息精简，一目了然，不堆砌内容

## 技术栈

- **框架**: Next.js 14 (App Router)
- **UI**: Tailwind CSS
- **状态管理**: Zustand
- **图标**: Lucide React
- **AI**: Qwen-3-235B
